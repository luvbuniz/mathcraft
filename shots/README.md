# 📸 Homepage screenshots

Drop your gameplay screenshots in **this folder** with these **exact filenames**, and
they'll appear in the "See it in action" gallery on the home page automatically. Until a
file exists, an on-brand placeholder (gradient + emoji + caption) shows instead — so the
page never looks broken.

| Filename | What to capture | Caption shown |
|---|---|---|
| `world.jpg` | A bright wide shot of the 3D world with the avatar | Explore a bright 3D world |
| `math.jpg` | A math question with the chunky answer blocks | Solve math → earn blocks |
| `drive.jpg` | Driving the **fire truck** (or police car) — lights on | Drive the fire truck — sirens & lights |
| `build.jpg` | A decorated house the kids built (the pink house!) | Build & decorate your own house |
| `cinema.jpg` | The 🎬 Grammar Cinema (movie pick or premiere) | Grammar Cinema: fix the script |
| `dashboard.jpg` | The grown-ups progress dashboard | Parents see real progress |

## Tips
- **Landscape** shots look best here (the tiles are 16:10). Capture on a tablet in
  landscape, or crop.
- **JPG**, reasonably compressed (aim < ~400 KB each) so the page stays fast.
- Bright, clear, no clutter. Real gameplay > staged.
- Want different shots? Swap any image — or tell Claude to change the filenames/captions.

## How to add them
- Easiest: on GitHub, open this `shots/` folder → **Add file → Upload files** → drag your
  images in (named as above) → commit. They go live on the next deploy.
- Or hand them to Claude and they'll wire them in.
