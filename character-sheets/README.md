# 🎨 Character Reference Sheets (canonical designs)

The locked, source-of-truth designs for the Stackadoo cast. Use these as the
reference image in Kling/Gemini for every cartoon, AND they're what the in-game
avatars are matched to. Consistency here = the cartoon→game bridge.

**This folder is kept OUT of the public site** (listed in `.assetsignore`), but
stays version-controlled on GitHub so the designs are never lost.

## How to add the actual sheet images
On github.com, open this `character-sheets/` folder → **Add file → Upload files**
→ drag in the JPGs. Suggested names: `fiona.jpg`, `leo.jpg`, `spark.jpg`.

## Canonical designs (text reference — keep in sync with the avatars)

- **Leo** 👦 — messy brown hair, **bright yellow t-shirt**, **blue shorts**,
  white-and-blue sneakers. Restless, cheeky, kind. *(In-game avatar: "Leo")*
- **Fiona** 👧 — brown shoulder-length wavy hair, **purple dress** with a
  **white Peter-Pan collar**, purple shoes. Clever, confident striver.
  *(In-game avatar: "Fiona")*
- **Spark the Fairy** ✨ — **lavender hair**, lavender top + **pastel petal
  skirt** (mint/pink/lilac), **sparkly blue-and-lavender wings**, a little glowing
  wand. Warm, magical guide. *(In-game avatar: "Spark the Fairy" — replaced the
  old generic Fairy)*
- **Murk the Fog Dragon** 🐉 (villain, held for a later cartoon episode) — dark
  gray-purple dragon, sickly-green eyes, breathes gray "muddle fog," shape-shifts
  into the swindler-of-the-week.
- **Mom** 🐼/voxel — warm, watchful; **Ice Cream Man** — towering, greasy-grinned
  human cheat (Episode 1 villain).
